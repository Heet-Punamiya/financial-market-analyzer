Theory: 
Statistical dispersion describes the extent to which data values in a dataset are 
spread or scattered around a central value. While measures of central tendency 
such as mean, median, and mode provide information about the central 
position of the data, measures of dispersion explain how much the data values 
differ from one another and from the central value. 
Understanding dispersion is important because datasets having the same mean 
can still differ greatly in terms of how their values are distributed. Therefore, 
analyzing dispersion helps in gaining deeper insights into the dataset. 
Visualization plays an important role in understanding statistical dispersion 
because graphical methods make it easier to observe patterns, spread, and 
variability in the data. 
Visualization of dispersion helps in: 
 Understanding the degree of variability present in the dataset 
 Identifying outliers or unusual values 
 Comparing the distribution of multiple datasets 
 Observing the shape and skewness of data distribution 
 Supporting better analysis and data-driven decision making 
Visualization Techniques for Dispersion in Python 
1. Histogram 
A histogram is a graphical representation that shows the frequency distribution 
of numerical data. It divides the data into different intervals (bins) and displays 
the number of observations in each interval. 
 Helps in understanding the distribution and spread of data 
 Shows whether the data is normally distributed, skewed, or uniform 
 Useful for observing variability and concentration of values 

--- Page 21 ---
2  
2. Box Plot 
A box plot (also called a box-and-whisker plot) is used to display the 
distribution of data based on five summary statistics. 
These include: 
 Minimum value 
 First Quartile (Q1) 
 Median (Q2) 
 Third Quartile (Q3) 
 Maximum value 
The box plot clearly shows the Interquartile Range (IQR) and helps in 
identifying outliers that fall outside the normal range of the data. 
3. Scatter Plot 
A scatter plot is used to represent the relationship between two numerical 
variables by plotting data points on a two-dimensional graph. 
 Each point represents one observation 
 The spread of points indicates the degree of dispersion 
 It is useful for identifying patterns, relationships, or correlations between 
variables 
Measures of Statistical Dispersion 
(a) Range 
Range is the simplest measure of dispersion and represents the difference 
between the highest and lowest values in a dataset. 
Range = Maximum Value − Minimum Value 
 Easy to calculate 
 Gives a quick idea of the overall spread of data 
 However, it is highly sensitive to outliers 
(b) Variance 

--- Page 22 ---
3  
Variance measures the average of the squared differences between each data 
value and the mean of the dataset. 
 It indicates how far the data values are spread from the mean 
 A larger variance means the data is more widely distributed 
 The unit of variance is expressed in squared form of the original data unit 
(c) Standard Deviation 
Standard deviation is the square root of the variance and represents the 
average distance of data points from the mean. 
 It is the most commonly used measure of dispersion 
 Provides a clear idea of how much the data deviates from the mean 
 Expressed in the same unit as the original dataset 
(d) Interquartile Range (IQR) 
The Interquartile Range measures the spread of the middle 50% of the dataset. 
IQR = Q3 − Q1 
Where: 
 Q1 is the first quartile (25th percentile) 
 Q3 is the third quartile (75th percentile) 
 It focuses only on the central portion of the dataset 
 Less affected by extreme values or outliers

Aim: Implementation of ID3 algorithm 
 
Theory: 
Implementation of the ID3 Algorithm – Theory 
The ID3 (Iterative Dichotomiser 3) algorithm is a popular decision tree learning technique 
used for classification tasks in data mining and machine learning. Developed by Ross 
Quinlan, ID3 builds a decision tree by selecting the attribute that best separates the data into 
different classes using the concept of Information Gain, which is based on Entropy from 
information theory. 
Basic Idea 
The main goal of the ID3 algorithm is to create a decision tree that can accurately classify 
data instances. It does this by recursively selecting the attribute that provides the maximum 
Information Gain, meaning the attribute that best reduces uncertainty about the target class. 
The process continues until all data instances are classified or no further useful splits are 
possible. 
Key Concepts 
1. Entropy 
Entropy measures the impurity or randomness in a dataset. If all instances belong to the same 
class, the entropy is 0 (pure). If the dataset contains a mix of classes, the entropy is higher. 
Formula for Entropy: 
[ 
Entropy(S) = - \sum_{i=1}^{n} p_i \log_2(p_i) 
] 
where ( p_i ) is the probability of instances belonging to class ( i ). 
2. Information Gain 
Information Gain measures the reduction in entropy achieved by splitting the dataset based 
on a particular attribute. The attribute with the highest Information Gain is selected for 
splitting. 
Formula for Information Gain: 

--- Page 42 ---
 
 
[ 
Gain(S, A) = Entropy(S) - \sum_{v \in Values(A)} \frac{|S_v|}{|S|} \times Entropy(S_v) 
] 
where: 
 
 ( S ) is the dataset, 
 
 ( A ) is the attribute, 
 
 ( S_v ) is the subset of ( S ) where attribute ( A ) takes value ( v ). 
Working of the ID3 Algorithm 
The implementation of ID3 follows a top-down, greedy, and recursive approach: 
1. 
 Start with the Entire Dataset 
The full training dataset is considered as the root of the decision tree. 
2. 
 Check for Pure Class 
o 
 If all instances belong to the same class, create a leaf node with that class 
label. 
o 
 If there are no remaining attributes to split, create a leaf node with the majority 
class. 
3. 
 Calculate Entropy of the Dataset 
Determine the impurity of the current dataset using the entropy formula. 
4. 
 Compute Information Gain for Each Attribute 
For every attribute, calculate the expected reduction in entropy after splitting the 
dataset. 
5. 
 Select the Best Attribute 
Choose the attribute with the highest Information Gain as the decision node. 
6. 
 Create Branches for Each Attribute Value 
Split the dataset into subsets based on the selected attribute’s possible values. 
7. 
 Recursively Apply the Same Process 
Repeat the above steps for each subset until one of the stopping conditions is met. 
Stopping Conditions 
The recursive splitting stops when: 
 
 All instances in a subset belong to the same class. 
 
 There are no remaining attributes to split. 
 
 The subset of data is empty, in which case the majority class of the parent node is 
assigned. 
 
 Information Gain becomes zero, indicating no further useful splits. 
Algorithm Steps (Pseudo-Code) 
ID3(S, Attributes, Target) 
 

--- Page 43 ---
 
 
1. If all examples in S belong to the same class: 
       Return a leaf node with that class. 
 
2. If Attributes is empty: 
       Return a leaf node with the majority class in S. 
 
3. Select attribute A with the highest Information Gain. 
 
4. Create a decision node with attribute A. 
 
5. For each value v of attribute A: 
       a. Create a branch for value v. 
       b. Let Sv be the subset of S where A = v. 
       c. If Sv is empty: 
              Attach a leaf node with the majority class of S. 
          Else: 
              Attach the subtree ID3(Sv, Attributes − {A}, Target). 
 
6. Return the decision node. 
Characteristics of ID3 
 
 Greedy Approach: Chooses the best attribute at each step without reconsidering 
previous decisions. 
 
 Top-Down Construction: Begins at the root and recursively builds the tree downward. 
 
 Handles Categorical Attributes: Works best with discrete data. 
 
 No Pruning: The basic ID3 algorithm does not include pruning, which may lead to 
overfitting. 
 
 Interpretable Model: Produces easy-to-understand decision rules.

Introduction to Clustering 
Clustering is an unsupervised learning technique used in data mining and machine learning 
to group similar data points together based on their features. In clustering, there are no 
predefined labels, and the algorithm tries to find patterns or structures within the data. 
Objects in the same cluster are more similar to each other than to objects in other clusters. 
Clustering is widely used in data analysis, pattern recognition, image processing, and 
business intelligence for grouping similar records. 
One of the most popular clustering algorithms is K-Means clustering, which partitions the 
dataset into a fixed number of clusters. To determine the correct number of clusters, the 
Elbow Method is used. 
K-Means Clustering 
K-Means clustering is a partition-based clustering algorithm that divides a dataset into K 
number of clusters, where K is given by the user. Each cluster has a centroid, which 
represents the center of that cluster. The algorithm assigns each data point to the nearest 
centroid and updates the centroid until the clusters become stable. 
Working of K-Means Algorithm 
1. Choose the number of clusters K. 
2. Select K initial centroids randomly from the dataset. 
3. Calculate the distance of each data point from all centroids. 
4. Assign each point to the nearest centroid. 
5. Recalculate the centroid of each cluster by taking the mean of points. 
6. Repeat steps 3–5 until centroids do not change or maximum iterations reached. 
This process minimizes the distance between data points and their cluster centers. 
Advantages of K-Means 
 Simple and easy to implement 
 Fast for small datasets 
 Works well when clusters are clearly separated 
 Efficient for large datasets 
Disadvantages of K-Means 

--- Page 51 ---
2 
 
 
 Need to specify number of clusters beforehand 
 Sensitive to initial centroid selection 
 Not suitable for irregular shaped clusters 
 Results may change due to randomness 
Elbow Method 
The Elbow Method is used to determine the optimal number of clusters in K-Means 
clustering. Choosing the correct value of K is important because too few clusters give poor 
grouping and too many clusters create unnecessary divisions. 
The elbow method uses WCSS (Within Cluster Sum of Squares) to measure cluster quality. 
At first, WCSS decreases rapidly. 
After some value of K, decrease becomes slow. 
That bending point is called the elbow. 
Applications of K-Means Clustering 
 Customer segmentation 
 Image compression 
 Document classification 
 Market analysis 
 Pattern recognition 
 Recommendation systems
